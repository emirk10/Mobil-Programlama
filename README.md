# Odaklanma Takibi ve Raporlama Uygulaması

**Ders:** BSM 447 - Mobil Uygulama Geliştirme
**Teknoloji:** React Native (Expo)

## Proje Hakkında

Bu proje, kullanıcıların belirli sürelerde odaklanmasını sağlamak ve dikkat dağınıklığını ölçmek amacıyla geliştirilmiştir. Uygulama, başlatılan sayaç süresince kullanıcının uygulamadan çıkıp çıkmadığını (AppState) takip eder. Eğer kullanıcı uygulamadan ayrılırsa, sayaç durur ve bu durum bir "dikkat dağınıklığı" olarak kaydedilir. Ayrıca tamamlanan seanslar cihaz hafızasında saklanır ve raporlar ekranında listelenir.

## Kurulum ve Çalıştırma

Projeyi çalıştırmak için bilgisayarınızda Node.js yüklü olmalıdır.

1. Proje klasörüne terminal üzerinden girin.

2. Gerekli paketleri yüklemek için şu komutu çalıştırın:

npm install


3. Uygulamayı başlatmak için şu komutu çalıştırın:

npx expo start


4. Ekranda çıkan QR kodu telefonunuzdaki Expo Go uygulaması ile okutarak projeyi test edebilirsiniz.
